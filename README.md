# Bonnie

One-page website for a relationship coaching practice. Direct, warm, anti-cliché — built on Relational Life Therapy (RLT) and the RESET framework.

## Status

**Project phase:** V1 build complete — content strategy implemented, ready for visual review.

## What This Is

A single-page site for a relationship coach. The strategy is grounded in Made to Stick (SUCCESs framework) and The Challenger Sale methodology. The core reframe: "You're not broken. You're stuck in a pattern."
